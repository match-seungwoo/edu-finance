### Step 5. 일일 수익률 분포 → Historical VaR

#### Q5-1. Historical VaR을 "가장 정직한 방법"이라 부르는 이유는?
**Check:** 분포 가정 없음 / 과거 데이터의 5% 분위수를 그대로 사용 / `np.percentile(port_ret, 5)` 한 줄 / 데이터가 보여주는 꼬리를 그대로 반영.

#### Q5-2. "VaR 95% = -2.4%"의 정확한 해석은? "최악의 손실이 -2.4%"라고 해석하면 왜 틀린가요?
**Check:** "정상적인 100거래일 중 95일은 -2.4%보다 덜 잃는다" — 나머지 5일은 **더** 잃을 수 있다 / VaR은 **경계선**이지 **한계**가 아님 / 꼬리 너머의 손실 크기는 CVaR이 답한다.

#### Q5-3. Historical VaR의 가장 큰 약점은?
**Check:** 과거에 없던 시나리오는 반영 못 함 / 데이터 양 의존(짧으면 분위수 추정 불안정) / 시간가중 없음(10년 전 충격과 어제 충격을 동일 취급).

### Step 6. CVaR (Expected Shortfall)

#### Q6-1. CVaR을 한 줄 코드로 표현하면?
**Check:** `cvar_95 = port_ret[port_ret <= var_95_hist].mean()` / VaR 임계 이하 수익률의 평균.

#### Q6-2. VaR과 CVaR이 같은 값을 가진 자산은 어떤 분포일까요?
**Check:** 임계 이하 영역에 단 하나의 값만 있는 분포(이론적) — 사실상 불가 / 일반적으로 |CVaR| ≥ |VaR| / 둘이 가까우면 꼬리가 얇음, 멀면 꼬리가 두꺼움.

#### Q6-3. "진짜 위험은 CVaR에서 드러난다"는 말의 의미는?
**Check:** VaR은 경계선만 본다 → 그 너머가 -3%인지 -30%인지 모름 / CVaR은 꼬리 평균 → "최악의 5% 평균 손실"을 정량화 / 같은 VaR이라도 CVaR이 다를 수 있음.

### Step 7. Parametric VaR (정규분포 가정)

#### Q7-1. Parametric VaR 공식과 그 안의 `-1.645`는 어디서 왔나요?
**Check:** `VaR_95% = μ + σ × norm.ppf(0.05)` / `norm.ppf(0.05) ≈ -1.645` = 표준정규분포의 5% 분위수 / 99%는 `-2.326`.

#### Q7-2. Parametric VaR이 Historical VaR보다 **작게(덜 보수적으로)** 나오는 일반적 이유는?
**Check:** 실제 수익률은 **fat tail** — 첨도 > 3 / 정규분포는 꼬리를 얇게 추정 / -1.645σ가 실제 5% 분위수보다 작음(즉 더 자주 깨짐).

#### Q7-3. Parametric의 장점은 무엇이며, 어떤 상황에서 유용한가요?
**Check:** 닫힌 공식 → 빠름 / 데이터 적어도 μ, σ 둘만 있으면 추정 / 신뢰구간·분석적 미분 가능 / 다자산 포트의 분산공분산법 기초.

### Step 8. Monte Carlo VaR

#### Q8-1. `np.random.seed(42)`를 쓴 이유는?
**Check:** 재현성 — 동일 시드로 다시 돌려도 같은 결과 / 수업·면접·디버깅 시 결과 비교 가능 / 실전에선 시드 고정과 다중 seed 평균을 함께 본다.

#### Q8-2. 단순 정규분포 MC는 Parametric VaR과 결과가 거의 같다. 그렇다면 MC를 쓰는 진짜 이유는?
**Check:** 다자산·비선형 상품(옵션, 구조화)에서 닫힌 공식이 없음 / 분포를 정규 외(Student-t, 혼합) 자유롭게 / 경로 의존 손익(barrier, MDD) 시뮬레이션 가능 / 단순 케이스에서 일치하는 건 "검증" 용도.

#### Q8-3. MC 표본 수 N을 1만에서 10만으로 늘리면 VaR 추정은 어떻게 변하나요?
**Check:** 분위수 추정 표준오차 ↓ (√N에 반비례) / 결과가 더 안정 / 다만 비용·시간 증가 / 비선형 경로 의존 상품에서 효과 큼.

### Step 10. 분산효과: 포트 VaR vs 종목별 VaR 합

#### Q10-1. "비중가중 단순 합산 VaR"과 "실제 포트 VaR"의 차이를 무엇이라 부르며, 부호는?
**Check:** 분산효과(diversification benefit) / `포트 VaR - 단순합` > 0이면 분산 효과 있음 / 종목 간 ρ < 1일 때 항상 발생.

#### Q10-2. 분산효과의 수학적 근거는?
**Check:** $\sigma_p^2 = \sum w_i^2 \sigma_i^2 + 2 \sum_{i<j} w_i w_j \sigma_i \sigma_j \rho_{ij}$ / ρ < 1이면 두 번째 항이 합을 줄임 / VaR도 σ에 비례하므로 같은 원리.

#### Q10-3. 두 종목이 완전 양의 상관(ρ=1)이라면 분산효과는 어떻게 되나요?
**Check:** 분산효과 = 0 / 포트 VaR = 비중가중 합 / 동일 섹터 5종목으로 "분산했다"고 착각하지 말 것.

### Step 15. 다요인 민감도 행렬

#### Q15-1. 민감도 행렬 × 충격 벡터 식을 일반화 형태로 쓰면?
**Check:** $\text{PnL} = B \cdot s$ — 종목 PnL 벡터 = (N×K 민감도 행렬) × (K차 충격 벡터) / 포트 PnL = $w^T B s$ / Step 13의 β는 B 행렬의 한 열일 뿐.

#### Q15-2. 노트북에서 `rate`, `fx_usd` 민감도를 **가정값**으로 둔 이유는? 실전에선 어떻게 추정하나요?
