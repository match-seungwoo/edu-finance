## ⚠️ 자주 터지는 에러 TOP 5

### 1. `ModuleNotFoundError: No module named 'kaleido'`

**증상:** `fig.write_image("chart.png")`에서 죽음.

**원인:** Plotly를 이미지로 저장하려면 `kaleido`가 따로 필요.

**해결:**
```python
!pip install kaleido -q
```

설치 후 **런타임 재시작 없이** 바로 동작한다. 학생이 설치 셀을 그냥 건너뛴 경우가 대부분.

---

### 2. `pykrx`가 빈 데이터프레임 반환

**증상:** `df`가 비었는데 에러는 안 남.

**원인 후보:**
- 주말/공휴일 범위로 끝나는 경우 → 날짜는 평일로
- 티커 오타 (앞자리 0 누락. `"5930"`이 아니라 `"005930"`)
- KRX 서버 순간 응답 불량 → 한 번 더 실행

**해결:**
```python
# 항상 6자리 문자열로 쓰기
ticker = "005930"  # ✅
ticker = 5930      # ❌ (정수)
ticker = "5930"    # ❌ (0 누락)
```

---

### 3. OpenAI API 401 Unauthorized

**증상:** `AuthenticationError: Incorrect API key provided`

**원인 후보:**
- 키에 공백/줄바꿈 섞임 (복붙 사고)
- 키 잔액($5 크레딧) 소진
- Colab Secret 이름 오타 (`OPENAI_API_KEY` vs `OPEN_API_KEY`)

**해결:**
```python
print(OPENAI_API_KEY[:7], len(OPENAI_API_KEY))
# "sk-proj" 로 시작하고 길이가 100 넘으면 정상
```

---

### 4. AI가 한국어 안 씀 / 형식 파괴

**증상:** system_prompt 무시하고 영어로 답하거나, 형식이 들쑥날쑥.

**원인:** `gpt-4o-mini`는 지시 따르는 힘이 약함. 비용 아끼려다 교육효과 떨어짐.

**해결:** 1주차는 **반드시 `gpt-4o`** 로. 비용 차이 얼마 안 난다 (수업당 $1 미만).

---

### 5. 한국어 CSV 깨짐

**증상:** 저장한 CSV를 엑셀로 열면 한글이 `��` 로 깨짐.

**해결:**
```python
summary.to_csv("...", encoding="utf-8-sig", index=False)
#                              ^^^^^^^ BOM 포함
```

## 🎤 학생이 자주 하는 질문 FAQ

### Q. "AI 의견 믿고 실제로 사도 돼요?"

**A.** 단호하게 NO.

```
"AI는 차트 하나만 봤어요. 재무제표, 뉴스, 거시경제 아무것도 모릅니다.
오늘 배우는 건 '도구 사용법'이지 '투자 신호'가 아니에요.
실제로 돈 넣기 전에는 최소 4주차 스트레스 테스트까지는 하고 결정하세요."
```

### Q. "왜 `matplotlib` 안 쓰고 `plotly`예요?"

**A.** 5~6주차 Streamlit/React로 갈 때 **코드 한 줄 안 바꾸고** 웹에서 그대로 뜬다. 오늘 투자한 시간이 나중에 회수된다.

### Q. "이동평균선 5/20/60 말고 다른 걸 보고 싶어요."

**A.** 대환영. `rolling(window=N)`의 N만 바꾸면 됨. 숙제로 내주면 좋다.

### Q. "미국 주식으로 하고 싶어요."

**A.** `pykrx` → `yfinance`로 갈아끼우면 끝.

```python
import yfinance as yf
df = yf.download("NVDA", period="1y")
# 컬럼명이 Open/High/Low/Close/Volume (영어)로 바뀜에 주의
```

### Q. "AI가 이평선을 무시하고 엉뚱한 얘기해요."

**A.** 세 가지 시도:
1. `scale=2` → `scale=3`으로 해상도 올리기
2. system_prompt에 "반드시 차트에 그려진 색깔 선(주황/파랑/빨강)을 이평선으로 인식해" 명시
3. 이미지에 텍스트 주석 추가 (`plotly`의 `annotations`)
