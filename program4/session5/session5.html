9차원 일괄 적용 — 한 줄 함수

# 6차원 툴킷 한 줄 import — 바퀴를 다시 만들지 않는다 from diplo_analysis import analyze_document, silence_map, mutuality_index # 라이브 분석 — 147개 문서 전부 (한 줄 함수가 6차원을 다 계산) rows = [analyze_document(d, nlp) for d in g] ana = pd.DataFrame(rows) # S4 교훈: 길이 정규화 — 강도동사를 단어수로 나눠 밀도로 ana["verb_density"] = ana["n_strength_verbs"] / ana["n_words"] * 1000

S2~3에서 손으로 짠 함수들이 이제 한 줄. spaCy 구문분석이라 라이브로 30초~1분 — 툴킷이 진짜 도는 걸 보여준다.

07

읽기 — 일관 vs 유연, 그리고 Q2

중국은 우크라이나에선 단독 1위, 가자에선 한국과 동반 최상위. 주제가 바뀌어도 늘 높다 = 앵커.

왜 중국은 두 주제 다 상호성이 높을까? 중립이라서? 당사자가 아니라서? 한국은 왜 가자에서만 폭증할까? — 이 물음이 곧 당사자성(Q2) 이다.

평균값은 '누가 그렇게 말하는가'까지만 보여준다. 언제부터, 어떻게 변했는가 는 시간을 봐야 한다 → 다음 세션.
