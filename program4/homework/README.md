## ▶️ 진행 방법

1. `ai_governance_assignment.ipynb` 를 Colab 또는 로컬 Jupyter에서 연다.
2. 셀을 **위에서 아래로 하나씩** 실행한다.
3. `# TODO` 가 보이면 빈칸(`____`)을 직접 채운다. 막히면 셀 아래 `💡 힌트`(`<details>`)를 펼친다.
   - 힌트는 *방향만* 준다. 전체 정답 코드는 정답본 노트북에 있다(먼저 보지 말 것).
4. 각 TODO 바로 아래 `# CHECK` 셀을 실행해 **`✅ PASS`** 가 떠야 다음으로 간다.
5. `📝 서술형` 마크다운 셀(Step 6)은 **너의 문장으로** 채운다(채점 대상).
6. 마지막에 **Runtime → Restart and run all** 로 처음부터 끝까지 에러 없이 도는지 확인한다.

> Claude(LLM) 확장 셀은 **선택·가산점**이다. API 키가 없으면 자동으로 `DEMO_MODE` 로 건너뛴다 —
> 규칙 기반 분석만으로 과제는 100% 완료된다.

## 🧮 채점 루브릭 (총 100점)

| 항목 | 배점 | 무엇을 보나 | 만점 기준 |
|---|---:|---|---|
| **코드 정확성** | 30 | 모든 `# CHECK` 셀 `✅ PASS`, 빈칸을 올바르게 채움, Restart-and-run-all 무에러 | CHECK 5개 전부 PASS + 재실행 에러 0 |
| **침묵지도 발견** | 20 | Step 2에서 **FR=0(구조적 부재)** 를 정확히 식별 | FR이 *8개 사건 전부*에서 부재임을 표로 보이고, 우발적 누락이 아님을 지적 |
| **비교분석 (당사자성)** | 20 | Step 3·4에서 소스 비교표 + 고/저 당사자성 비교를 올바르게 구성, **밀도 정규화** 적용 | AI 상호성↑·동사밀도↓ 패턴을 *밀도(verb_density)* 근거로 제시 |
| **서술형 통찰** | 20 | Step 6 (a)(b)(c) — 숫자 근거 + 측정 한계 인지 | (a) FR 부재의 *구조적* 이유, (b) 수렴/발산 판단+근거, (c) 한계 2가지(특히 n 작음·FR 부재) |
| **시각화** | 10 | Step 5 차트가 데이터를 올바르게·읽기 쉽게 전달 | 제목·축 라벨·값 표시가 갖춰진 차트 한 장 이상 |

**감점/주의**
- `verb_strength_max`(원시 최댓값)만 보고 "중국이 강경"이라 결론 → **길이 편향**을 놓침(−). S4의 밀도 교훈을 적용할 것.
- 표에 FR 행을 억지로 만들려다 에러 → FR은 *데이터가 없는 게 정답*이다.
- n=21 + FR 부재를 잊고 인과·일반화로 단정 → '경향'까지만 허용.
- 서술형을 숫자 근거 없이 인상비평으로 채움 → 통찰 점수 대폭 감점.

**가산점 (+α)**
- Claude 의미 분석 확장(`DEMO_MODE` 가드)을 실제 키로 돌려 framing/미묘한 hedging을 규칙 점수와 교차검증.
- Step 5에서 예시(막대) 대신 레이더/산점도 등으로 *분석 의도가 더 잘 드러나는* 차트를 설계.

## 🔑 핵심 발견 (참고 — 정답본에 수치 포함)

- **프랑스(FR)는 AI 거버넌스에서 0건** — 우발적 누락이 아니라 **구조적 부재**다.
  프랑스의 AI-정상회의 자료(파리 AI Action Summit 등)는 외교부 성명 트리가 아니라
  **엘리제궁/정상회의 전용 포털**에 살아서, 우리 스크래퍼의 사각지대에 있었다.
- **당사자성이 낮을수록 언어가 부드러워진다:** AI(저당사자성)는 상호성이 우크라이나·가자(고당사자성)의
  약 2배, 동사강도 밀도는 약 절반이다. 직접 이해가 안 걸린 주제라 각국이 *협력 어휘로 수렴*하는 경향.

> 이 발견들은 **작은 표본(21건) + 프랑스 부재**라는 한계 위에 있다. 결론은 *경향* 수준으로만 말한다.
